BetterTeams

Create teams to fight to be the best (Minecraft Plugin)

Plugins that work with BetterTeams:

This plugin supports many other plugins. Here is a list of other plugins BetterTeams can work with and how they work together:

Placeholder Api:

This plugin supports many placeholders which can be used through placeholder API. For more information view the placeholderAPI page here

Ultimate Claims:

This plugin supports altering how the claiming system of UltimateClaims works. For more information view the UltimateClaims page of the wiki here

Vault:

A team can have a balance associated with their team, this is managed with the commands:

/team deposit

/team bal [team]

/team baltop

A team can have a balance associated with their team, this is managed with these commands.

Holographic displays:

This plugin allows for the creation of holographic leaderboards which can either order the teams by score or by team balance. Commands associated with this integration are

/teama holo create >score/money< - Create a holographic leaderboard of that type where you are standing

/teama holo remove - Remove the closest BetterTeams managed hologram to you

zKoth:

Players can win a configurable amount of score for their team when winning king of the hill events. This amount is configurable in the config.yml file under zKoth pointsPerCapture heading.

Turrets:

Turrets do not attack allies

Worldguard:

There is a WorldGaurd flag called teamPvp which when set to allow, will let players of the same team hit each other within that region.
